# A股财报预期对比分析工具

## 🚨 强制规则
按照工作流程使用AKShare函数获取数据，严格按照对比逻辑进行分析，确保输出格式清晰规范。
不要通过web search等方式，会导致数据不准确或过时。

## 功能说明

本skill用于对比A股公司发布的财报实际数据与机构预测数据，判断业绩是否符合预期。

**核心功能：**
- 获取股票的实际财报数据（EPS、营收、净利润等）
- 获取机构预测数据（基于分析师研报）
- 自动对比分析，判断是否符合预期
- 生成清晰的对比报告

**判断标准：**
- 差异率 > 5%：超预期
- -5% ~ 5%：符合预期
- 差异率 < -5%：不及预期

**数据源：**
- 实际财报：AKShare（东方财富）
- 机构预测：AKShare（东方财富研报）

## 工作流程

### 第一步：获取财报发布清单（可选）

当需要查看某日发布财报的公司清单时使用。

**AKShare函数：**
```python
ak.stock_notice_report(symbol='全部', date='YYYYMMDD')
```

**参数说明：**
- `symbol`: 固定为 '全部'
- `date`: 日期，格式 YYYYMMDD，如 '20260402'

**返回数据：**
- 包含所有在该日期发布公告的公司
- 需筛选包含"年报"或"年度报告"标题的记录

**输出示例：**
```
✓ 找到 31 家公司发布年报

统计信息：
  总数：31 家
  净利润增长：20 家
  净利润下降：11 家
```

### 第二步：获取实际财报数据

获取指定股票的最新财报数据。

**AKShare函数：**
```python
ak.stock_financial_abstract(symbol='股票代码')
```

**参数说明：**
- `symbol`: 6位股票代码，如 '000001'

**返回数据字段：**
| 字段 | 说明 |
|------|------|
| 基本每股收益 / 每股收益 | EPS |
| 营业总收入 / 营业收入 | 营业收入 |
| 净利润 | 净利润 |
| 净资产收益率(ROE) / 净资产收益率 | ROE |
| 毛利率 / 销售毛利率 | 毛利率 |

**数据处理：**
- 使用 `set_index('指标').T` 转置数据
- 取第一行（最新一期）作为当前财报数据
- 同比增长率需对比第5期数据计算

### 第三步：获取机构预测数据

获取分析师对该股票的业绩预测。

**AKShare函数：**
```python
ak.stock_research_report_em(symbol='股票代码')
```

**参数说明：**
- `symbol`: 6位股票代码，如 '000001'

**返回数据字段：**
- 盈利预测列：`{年份}-盈利预测-收益`（EPS预测）
- 营收预测列：`{年份}-盈利预测-营业总收入`
- 净利润预测列：`{年份}-盈利预测-净利润`
- `日期`：预测发布时间

**数据处理：**
- 提取对应年份的盈利预测列
- 计算预测均值、最小值、最大值
- 统计预测机构数量

### 第四步：对比分析

**对比逻辑：**
```python
差异率 = (实际值 - 预测值) / |预测值| × 100%

if 差异率 > 5%:
    判断 = '超预期'
elif 差异率 < -5%:
    判断 = '不及预期'
else:
    判断 = '符合预期'
```

**综合判断规则：**
- 任一指标超预期 → 超预期
- 任一指标不及预期 → 不及预期
- 全部符合预期 → 符合预期
- 无预测数据 → 数据不足

## 注意事项

### 数据可用性

1. **机构预测数据**：
   - 并非所有股票都有机构预测
   - 小盘股可能预测机构数量较少
   - 预测数据可能滞后

2. **财报数据**：
   - 使用最新发布的财报
   - 数据可能有延迟

### 判断标准

默认阈值：5%

### 数据时效性

- 预测数据会随时间更新
- 建议在财报发布后及时分析
- 注意预测日期与报告期匹配

## 常见问题

**Q: 为什么显示"数据不足"？**

A: 可能原因：
- 该股票无机构预测数据
- EPS数据缺失
- 财报数据未更新

**Q: 预测数据来自哪里？**

A: 来自东方财富网的研报数据，由分析师发布。
