# 数据字段说明

## 年报日历数据字段

| 字段名 | 中文名 | 说明 |
|--------|--------|------|
| SECURITY_CODE | 股票代码 | 6位数字代码 |
| SECURITY_NAME_ABBR | 股票简称 | 股票的简称 |
| REPORT_DATE | 报告日期 | 年报发布日期 |
| REPORT_TYPE | 报告类型 | 如：年报、半年报等 |

## 财务数据字段

| 字段名 | 中文名 | 单位 | 说明 |
|--------|--------|------|------|
| yysr | 营业收入 | 亿元 | 主营业务收入 |
| jlr | 净利润 | 亿元 | 归属母公司股东的净利润 |
| yysrtbzz | 营收同比增长 | % | 与去年同期相比的增长率 |
| jlrtbzz | 净利润同比增长 | % | 与去年同期相比的增长率 |
| mgjlr | 每股收益 | 元 | EPS |
| roe | 净资产收益率 | % | Return on Equity |
| xsmll | 销售毛利率 | % | 毛利率 |
| jll | 净利率 | % | 净利润率 |

## 机构预测数据字段

| 字段名 | 中文名 | 单位 | 说明 |
|--------|--------|------|------|
| REVENUE_FORECAST | 营收预测 | 亿元 | 机构预测的营业收入 |
| NET_PROFIT_FORECAST | 净利润预测 | 亿元 | 机构预测的净利润 |
| EPS_FORECAST | EPS预测 | 元 | 机构预测的每股收益 |
| REPORT_DATE | 报告日期 | - | 预测发布日期 |
| RESEARCH_INST | 研究机构 | - | 发布预测的机构名称 |

## 分析结果字段

| 字段名 | 中文名 | 说明 |
|--------|--------|------|
| code | 股票代码 | 6位数字 |
| name | 股票名称 | - |
| overall_judgment | 综合判断 | 超预期/符合预期/不及预期/无预测数据 |
| metrics | 指标列表 | 包含各指标的对比结果 |
| forecast_count | 预测机构数 | 提供预测的机构数量 |

## 指标对比字段

| 字段名 | 中文名 | 说明 |
|--------|--------|------|
| metric | 指标名称 | 如：营业收入、净利润等 |
| actual | 实际值 | 年报中的实际数值 |
| forecast | 预测值 | 机构预测的平均值 |
| diff_rate | 差异率 | (实际值-预测值)/预测值 × 100% |
| judgment | 判断结果 | 超预期/符合预期/不及预期 |
| unit | 单位 | 亿元/元/% 等 |

## 判断标准

| 差异率范围 | 判断结果 |
|------------|----------|
| > 5% | 超预期 |
| -5% ~ 5% | 符合预期 |
| < -5% | 不及预期 |
