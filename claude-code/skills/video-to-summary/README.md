# Video to Summary

视频转文字总结工具 - 支持 B站、YouTube、小红书、TikTok、Twitter 等平台。

## 快速开始

```bash
# 安装
bash install.sh

# 使用
python scripts/video_to_summary.py "视频URL"
```
